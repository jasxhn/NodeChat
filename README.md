# 💬 NodeChat

A modern real-time chat application built with the **MERN Stack**, featuring secure authentication, responsive UI, and instant messaging.

🌐 **Live Demo:** https://nodechat-bhzk.onrender.com
🌐 **Repo Link:** https://github.com/jasxhn/NodeChat

---

## ✨ Features

* 🔐 User Authentication
* 💬 Real-time Messaging
* 👤 User Profiles
* 🌙 Modern & Responsive UI
* ⚡ Fast React + Vite Frontend
* 🚀 Node.js & Express Backend
* 🗄️ MongoDB Database

---

## 🛠️ Tech Stack

### Frontend

* React
* Vite
* Tailwind CSS
* DaisyUI
* Axios

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication
* bcrypt

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/jasxhn/NodeChat.git
cd NodeChat
```

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file inside the `backend` folder.

Example:

```env
PORT=5001
MONGODB_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
CLOUDINARY_CLOUD_NAME=your_cloudinary_name
CLOUDINARY_API_KEY=your_api_key
CLOUDINARY_API_SECRET=your_api_secret
NODE_ENV=development
```
